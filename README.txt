Humble Game store
This is a asp.net Mvc Framework Web application for an online game store. 
A.Loading the Database
1.The database can be loaded using the db-6b.sln file. 
2.Change the data source ".....data source=DESKTOP-*******" in <connectionStrings> tag of App.config file to your local server. 
3.Create a HumbleDB database using mircosoft sql server management studio.
4.Copy the content of Model1.edmx.sql into SQL and execute it. 
5.Press "run" or f5 to seed the database.
B.Using the Database.
1.Open the  DB/Data.cs file and change the connectionString ="Server=DESKTOP-*******" to your local server.
2.click on IIS Express or f5 to use the web application.
